Control Modules
===============

Each :ref:`Equipment Modules <Equipment modules>` consists of one or more **Control Modules**. 

--------------------------------------------------------------------------------------

A Control module is a piece of software that controls the behaviour of one specific sensor or actuator

in short, an equipment module is:

- A collection of sensors, actuators, or other Control modules
- Is the direct connection to the process through its sensor and actuators

-------------------------------------------------------------------------------------

For example: Valve, Pump, Motor, Pressure Controller, Limit Switch

-------------------------------------------------------------------------------------
